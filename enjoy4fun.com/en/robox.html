About Robox

Robox is a puzzle platformer game where you control a robot to find the exit door. Help him to escape enemies and traps along the way. Jump, shoot the boxes using your teleportation gun, and try to reach the door. Good luck!

How to play Robox online?

A / left arrow key = move left D / right arrow key = move right W / Up Arrow = jump X / J = shoot R = reset Esc / B = back
