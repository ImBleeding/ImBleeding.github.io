About Link

Link is a puzzle game where you must connect the dots with the same color to solve each puzzle. Beware! Pipes can break if they criss-cross! Enjoy this addictive puzzle game!

How to play Link online?

Left-click = select a color. Move your mouse to move the link Arrow keys = move Enter = select Backspace = back I = hint
